# standup-form

- Blank Qwik starter app.
- Vite.js tooling.
- Eslint code linting.
- Prettier code formatter.
- Tailwind CSS framework.

## Development

### Install

```
npm install
```

### Development Build

```
npm run dev
```

### Production Build

```
npm run build
```
